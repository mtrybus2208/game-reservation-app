# Reservly
Reservly is a tool for reserving game match queue

## Frontend technology stack
* React
* Redux
* Saga
* Styled components

## How to run?
* Install [NPM](https://nodejs.org/en/download/) 
* Go to project root directory
* Run `npm install`
* Run `npm run start`
