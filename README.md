# Reservly
Reservly is a tool for reserving game match queue

## Frontend technology stack
* React
* Redux
* Styled components

## How to run?
* Download and install NPM (e.g. [here](https://nodejs.org/en/download/))
* Go to project root directory
* Run `npm install`
* Run `npm run start`
